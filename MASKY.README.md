# Input & Output – Three Mini Programs
This file contains three small programs that each demonstrate a different type of input/output behavior in Python.

## 1. Counter (1 to 10)
- Counts from 1 to 10 using a for‑loop and prints each number.

## Learning Purpose
- for‑loops
- range()
- simple numeric output




## 2. Reverse Word
- Asks the user for a word and prints it reversed using slicing.

## Learning Purpose
- user input
- string slicing ([::-1])
- basic output formatting




## 3. Word Collector
- Keeps asking the user for words until they type "stop".
- All words (except "stop") are saved in a list and printed at the end.

## Learning Purpose
- while‑loops
- breaking loops
- list building
- user input handling
